# K-pop US Analysis

This project explores why K-pop became highly successful in the United States.

## Topics

- Social media strategy
- YouTube growth
- Recommendation algorithms
- Fan culture

## Structure

- `data/` — raw and processed datasets
- `notebooks/` — analysis notebooks and exploratory work
- `scripts/` — reusable data processing and modeling scripts
- `reports/` — findings, visuals, and summaries
- `docs/` — supporting research notes and references

## Getting Started

1. Create a Python environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt
   ```
2. Open `notebooks/00-kpop-us-analysis.ipynb` for the first analysis workflow.
3. Add datasets to `data/raw/` or `data/` as needed.
